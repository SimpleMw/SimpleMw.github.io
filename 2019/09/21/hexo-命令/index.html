hexo 命令

Created2019-09-21|Updated2021-08-24|hexo相关
|Post View:

  • 清空资源
    hexo clean

  • 新建文章
    hexo new “文章名”

  • 新建页面

    hexo new page “页面名称”

  • 生成资源
    hexo generate （简写 hexo g）

  • 部署服务器
    hexo server （简写 hexo s）

  • 推送github
    hexo deploy

  • 下载主题
    git clone 主题github地址 themes/别名

注：下载主题至themes下的 别名文件下

  • hexo 图片文件插件

    npm install hexo-asset-image -save

  • 注：使用的Chic主题

    在第一次初始化时要创建category和tag

    hexo new page tag
    hexo new page category

    然后分别添加标签

    layout：tag

    layout：category
