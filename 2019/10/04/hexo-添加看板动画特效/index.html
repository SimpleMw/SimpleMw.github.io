添加看板动画特效

  • 安装依赖

    pm install –save hexo-helper-live2d

注：卸载依赖：npm uninstall hexo-helper-live2d

依赖可以在 主目录的 package.json中查看

  • 选择看板主题（黑猫看板主题）

    npm install live2d-widget-model-hijiki

    注：完成后，主目录的node_moduels文件夹下有相应的文件夹

  • 主目录修改配置文件
