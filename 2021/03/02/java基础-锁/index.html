жӮІи§Ӯй”ҒгҖҒд№җи§Ӯй”Ғ

  • жӮІи§Ӯй”Ғ

    • еҒҮи®ҫеңЁдҝ®ж”№ж•°жҚ®ж—¶пјҢеҗҢж—¶жңүеҸҰеӨ–зҡ„зәҝзЁӢдҝ®ж”№ж•°жҚ®
    • еӨ„зҗҶд№ӢеүҚеҠ й”ҒпјҢж•°жҚ®ж“ҚдҪңе®ҢжҲҗеҗҺйҮҠж”ҫй”Ғ

  • д№җи§Ӯй”Ғ

    • еҒҮи®ҫеңЁдҝ®ж”№ж•°жҚ®ж—¶пјҢдёҚдјҡжңүе…¶е®ғзҡ„зәҝзЁӢдҝ®ж”№ж•°жҚ®

    • еӨ„зҗҶдёҚеҠ й”ҒпјҢеҗҢжӯҘж•°жҚ®ж—¶иӢҘж•°жҚ®е·Із»Ҹиў«еҸҰеӨ–зәҝзЁӢдҝ®ж”№пјҢеҲҷеҒҡеӨұиҙҘжҲ–иҖ…йҮҚиҜ•еӨ„зҗҶ

иҮӘж—Ӣй”Ғ
  • иӢҘйҳ»еЎһзәҝзЁӢе”ӨйҶ’зәҝзЁӢзҡ„ејҖй”ҖеӨ§пјҢеҸҜе°ҶзәҝзЁӢиҝӣиЎҢеҫӘзҺҜзӯүеҫ…
  • е®һйҷ…жҳҜзүәзүІCPU(еӨ„зҗҶеҷЁж—¶й—ҙ)пјҢжҚўеҸ–зәҝзЁӢеҲҮжҚўзҡ„ејҖй”Җ

й—®йўҳпјҡиӢҘдёҖзӣҙж— жі•жҠўеҚ еҲ°иҝҗиЎҢжқЎд»¶пјҢдјҡдёҖзӣҙиҮӘж—Ӣпјӣи§ЈеҶіж–№жі•жҺ§еҲ¶иҮӘж—Ӣж¬Ўж•°

е…¬е№ій”ҒгҖҒйқһе…¬е№ій”Ғ
  • е…¬е№ій”ҒпјҡзәҝзЁӢжҢүз…§з”іиҜ·й”Ғзҡ„йЎәеәҸеӯҳеңЁйҳҹеҲ—дёӯпјҢеҸӘжңүйҳҹеҲ—з¬¬дёҖдёӘзәҝзЁӢиғҪиҺ·еҫ—й”ҒпјҢе…¶е®ғзәҝзЁӢйғҪжҳҜйҳ»еЎһ
  • йқһе…¬е№ій”ҒпјҡзәҝзЁӢд№ҹжҳҜжҢүз…§з”іиҜ·й”Ғзҡ„йЎәеәҸеӯҳеңЁйҳҹеҲ—дёӯпјҢдҪҶжҳҜеҲҡжқҘзҡ„зәҝзЁӢиӢҘзў°еҲ°жӯЈеҘҪжңүй”ҒеҸҜз”ЁеҲҷиғҪзӣҙжҺҘжҸ’еҲ—иҺ·еҫ—й”ҒпјҢжӯЈжүҖи°“жқҘеҫ—ж—©дёҚеҰӮжқҘеҫ—е·§

е…¬е№ій”ҒпјҡзјәзӮ№жҳҜйҳ»еЎһе’Ңе”ӨйҶ’зәҝзЁӢејҖй”Җж— жі•йҒҝе…ҚпјҢйқһе…¬е№ій”ҒеҜ№дёҺжқҘеҫ—е·§зҡ„зәҝзЁӢеҲҷж— иҜҘејҖй”Җ

йқһе…¬е№ій”ҒпјҡзјәзӮ№жҳҜжңүдәӣзәҝзЁӢеҸҜиғҪдёҖзӣҙиҺ·еҸ–дёҚеҲ°й”ҒпјҢжҲ–иҖ…зӯүеҫ…еҲ°й”Ғзҡ„ж—¶й—ҙеҫҲй•ҝ

еҸҜйҮҚе…Ҙй”ҒгҖҒдёҚеҸҜйҮҚе…Ҙй”Ғ
  • еҸҜйҮҚе…Ҙй”ҒпјҡеҗҢдёҖдёӘзәҝзЁӢеңЁеӨ–еұӮж–№жі•иҺ·еҸ–й”Ғзҡ„ж—¶еҖҷпјҢи°ғз”ЁиҜҘзәҝзЁӢзҡ„еҶ…еұӮж–№жі•дјҡиҮӘеҠЁиҺ·еҸ–й”ҒпјҢйҒҝе…ҚеҮәзҺ°жӯ»й”Ғжғ…еҶөпјҢеңЁеҲҶеёғејҸй”ҒдёӯжҳҜдјҡз”ЁдёҖдёӘеӯ—ж®өжқҘи®°еҪ•еҠ й”Ғзҡ„еұӮж•°
  • дёҚеҸҜйҮҚе…Ҙй”ҒпјҡдёҺеҸҜйҮҚе…Ҙй”ҒзӣёеҸҚпјҢеҗҢдёҖдёӘзәҝзЁӢеңЁеӨ–еұӮж–№жі•иҺ·еҸ–й”Ғзҡ„ж—¶еҖҷпјҢи°ғз”ЁиҜҘзәҝзЁӢзҡ„еҶ…еұӮж–№жі•дёҚдјҡиҺ·еҸ–еҲ°иҜҘй”Ғ

жіЁпјҡдёҚеҸҜйҮҚе…Ҙй”ҒзјәзӮ№пјҡз”ұеӨ–еұӮж–№жі•иҺ·еҸ–й”Ғзҡ„ж—¶еҖҷпјҢиӢҘеңЁиҜҘзәҝзЁӢдёӯжңүи°ғз”ЁиҮӘе·ұжң¬зәҝзЁӢж–№жі•зҡ„ж—¶еҖҷз”ұдәҺжң¬зәҝзЁӢжңӘйҮҠж”ҫпјҢеҲҷж— жі•и°ғз”Ёжң¬зәҝзЁӢж–№жі•пјҢеҜјиҮҙжң¬зәҝзЁӢж— жі•йҮҠж”ҫпјҢеҪўжҲҗжӯ»й”Ғ

зӢ¬дә«й”ҒгҖҒе…ұдә«й”Ғ
  • зӢ¬дә«й”ҒпјҡиҺ·еҸ–еҲ°й”ҒеҗҺпјҢжң¬зәҝзЁӢзӢ¬дә«иҜ»еҶҷ
  • е…ұдә«й”ҒпјҡиҺ·еҸ–еҲ°й”ҒеҗҺпјҢжң¬зәҝзЁӢиҜ»еҶҷпјҢе…¶е®ғзәҝзЁӢиҜ»
й”Ғзҡ„зҠ¶жҖҒ

ж— й”ҒгҖҒеҒҸеҗ‘й”ҒгҖҒиҪ»йҮҸзә§й”ҒгҖҒйҮҚйҮҸзә§й”Ғ

й”ҒеҸӘиғҪеҚҮзә§дёҚиғҪйҷҚзә§пјҢж–№еҗ‘дёә ж— й”ҒвҖ“>еҒҸеҗ‘й”ҒвҖ“>иҪ»йҮҸзә§й”ҒвҖ“>йҮҚйҮҸзә§й”Ғ

  • ж— й”ҒвҖ“>еҒҸеҗ‘й”Ғ еҪ“еҗҢжӯҘд»Јз ҒдёҖзӣҙиў«дёҖдёӘзәҝзЁӢеҚ з”ЁпјҢж— й”Ғе°ұдјҡиҪ¬дёәеҒҸеҗ‘й”Ғ
  • еҒҸеҗ‘й”ҒвҖ“>иҪ»йҮҸзә§й”Ғ еҪ“жңүеҸҰеӨ–зҡ„дёҖдёӘзәҝзЁӢеҠ е…ҘиҝӣжқҘпјҢеҒҸеҗ‘й”ҒдјҡиҪ¬дёәиҪ»йҮҸзә§й”ҒпјҢиҜҘеҠ е…Ҙзҡ„зәҝзЁӢдјҡе·ІиҮӘж—Ӣзҡ„ж–№ејҸиҝӣиЎҢеҫӘзҺҜзӯүеҫ…
  • иҪ»йҮҸзә§й”ҒвҖ“>йҮҚйҮҸзә§й”Ғ еҪ“еҸҲжңүдёҖдёӘзәҝзЁӢжҲ–еӨҡдёӘзәҝзЁӢеҠ е…ҘиҝӣжқҘпјҢиҪ»йҮҸзә§й”ҒдјҡиҪ¬дёәйҮҚйҮҸзә§й”ҒпјҢзӯүеҫ…й”Ғзҡ„зәҝзЁӢдјҡиҪ¬дёәйҳ»еЎһзҠ¶жҖҒ

ж“ҚдҪңзі»з»ҹзҡ„и°ғз”ЁеҲҶдёәеҶ…ж ёжҖҒе’Ңз”ЁжҲ·жҖҒ

  • ж— й”ҒгҖҒеҒҸеҗ‘й”ҒгҖҒиҪ»йҮҸзә§й”ҒжҳҜеңЁз”ЁжҲ·жҖҒиҝҗиЎҢзҡ„

  • йҮҚйҮҸзә§й”ҒжҳҜеңЁеҶ…ж ёжҖҒиҝҗиЎҢзҡ„

еҒҸеҗ‘й”ҒпјҡеҮҸе°‘еҜ№иұЎдёӢдёҖж¬Ўи®ҝй—®зәҝзЁӢж—¶еҶҚж¬ЎиҺ·еҸ–й”ҒиҝҮзЁӢ

иҪ»йҮҸзә§й”ҒпјҡеҪ“еҜ№иұЎзҡ„иҺ·еҸ–жңүз«һдәүж—¶пјҢзәҝзЁӢйғҪдјҡд»ҘCASиҮӘж—Ӣж–№ејҸеҺ»жҠўеҚ еҜ№иұЎпјҢжҠўеҲ°еҜ№иұЎзҡ„зәҝзЁӢдјҡе°ҶзәҝзЁӢidеӯҳеңЁ mark wordдёӯ

йҮҚйҮҸзә§й”ҒпјҡйҮҚйҮҸзә§й”ҒжҳҜеҶ…ж ёжҖҒзҡ„пјҢеҪ“иҮӘж—Ӣж¬Ўж•°иҫҫеҲ°и®ҫе®ҡзҡ„жңҖеӨ§иҮӘж—Ӣж¬Ўж•°пјҢдјҡеҚҮзә§дёәйҮҚйҮҸзә§й”ҒпјҢmark wordдёӯжҳҜжҢҮеҗ‘дә’ж–ҘйҮҸзҡ„жҢҮй’ҲпјҢйҮҚйҮҸзә§й”ҒжҳҜдә’ж–Ҙй”Ғ

жіЁпјҡиҪ»йҮҸзә§й”ҒвҖ“>йҮҚйҮҸзә§й”Ғ еҺҹеӣ пјҡзӯүеҫ…зҡ„зәҝзЁӢеӨӘеӨҡдәҶпјҢе°ҶжңӘиҺ·еҸ–еҲ°й”Ғзҡ„зәҝзЁӢж”ҫе…Ҙ зӯүеҫ…зҠ¶жҖҒ

й”Ғж¶ҲйҷӨ

иӢҘеҜ№иұЎдёӯеӯҳеңЁ еҠ дәҶsynchronizedеҗҢжӯҘеӯ—ж®өзҡ„ж–№жі•пјҢиӢҘjvmеҲӨж–ӯиҜҘеҜ№иұЎдёҚеҸҜиғҪиў«е…¶е®ғзәҝзЁӢеј•з”ЁпјҢеҲҷjvmдјҡиҮӘеҠЁдјҳеҢ–пјҢдјҡиҮӘеҠЁж¶ҲйҷӨеҜ№иұЎеҶ…йғЁзҡ„й”Ғ еҰӮпјҡStringBuffer

й”ҒзІ—еҢ–

иӢҘеҜ№иұЎдёӯеӯҳеңЁ еҠ дәҶsynchronizedеҗҢжӯҘеӯ—ж®өзҡ„ж–№жі•пјҢиӢҘиҜҘеҜ№иұЎиў«еӨҡж¬ЎеҠ й”Ғ(еҫӘзҺҜ)пјҢеҲҷjvmдјҡе°ҶеҠ й”Ғж“ҚдҪңзІ—еҢ–еҲ°еҫӘзҺҜеӨ–йғЁ

еҖҹйүҙеҮәеӨ„

synchronized

е®һзҺ°еҺҹзҗҶпјҡjavaеҺҹз”ҹең°е®һзҺ°й”ҒпјҢеҚіжҳҜзәҝзЁӢе°ҶзәҝзЁӢidеҶҷе…ҘеҲ°еҜ№иұЎең°markwordдёӯпјҢе“ӘдёӘзәҝзЁӢеҶҷе…ҘдәҶпјҢе°ұе“ӘдёӘзәҝзЁӢжҠўеҲ°дәҶй”Ғ

дёүдёӘеұӮйқўи§Јжһҗsynchronizedпјҡ

  • д»Јз ҒеұӮйқў еҚіsynchronizedдҝ®йҘ°ж–№жі•жҲ–иҖ…д»Јз Ғеқ—

  • еӯ—иҠӮз ҒеұӮйқў иў«synchronizedдҝ®йҘ°зҡ„д»Јз Ғеқ—пјҢеңЁеӯ—иҠӮз ҒеұӮйқўжҳҜ з”Ёmonitorenterе’ҢmonitorexitиҝҷдёӨдёӘе‘Ҫд»ӨжқҘе®һзҺ°еҠ й”Ғе’Ңи§Јй”Ғзҡ„

  • жұҮзј–еұӮйқў еҚіlockдҝ®йҘ°пјҢCASзҡ„еҺҹзҗҶеңЁжӯӨеӨ„зҡ„е®һзҺ°е‘Ҫд»Ө cmpxchgпјҢеҚіжҜ”иҫғж”№еҸҳпјҢlockжқҘдҝ®йҘ° cmpxchgдҝқиҜҒеҺҹеӯҗжҖ§
