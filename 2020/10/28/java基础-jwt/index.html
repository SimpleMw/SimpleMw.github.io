  • з»„жҲҗ

    • еӨҙйғЁ(header)

      еӨҙйғЁеӯҳ зұ»еһӢе’ҢеҠ еҜҶз®—жі•

    • жңүж•ҲиҪҪиҚ·(playload)

      е®һйҷ…йңҖиҰҒдҝқеӯҳзҡ„еҸӮж•°пјҢеҰӮtokenзӯҫеҸ‘иҖ…гҖҒзӯҫеҸ‘ж—¶й—ҙгҖҒеҲ°жңҹж—¶й—ҙпјҢд»ҘеҸҠеҰӮз”ЁжҲ·еҗҚд№Ӣзұ»зҡ„еҗҺз»ӯйңҖиҰҒз”ЁеҲ°зҡ„еҸӮж•°гҖӮ жіЁпјҡеҰӮеҜҶз Ғзӯүж¶үеҸҠеҲ°е®үе…Ёзҡ„дҝЎжҒҜдёҚе®ңж”ҫиҝӣеҺ»

    • зӯҫеҗҚ(signature)

      еӨҙйғЁе’Ңжңүж•ҲиҪҪиҚ·еҠ дёҠеҜҶй’ҘйҖҡиҝҮеӨҙйғЁйҮҢйқўзҡ„еҠ еҜҶз®—жі•иҝӣиЎҢеҠ еҜҶеҗҺеҫ—еҲ°

  • еӨҙйғЁдёҺжңүж•ҲиҪҪиҚ·еқҮйҖҡиҝҮbase64иҝӣиЎҢеҠ еҜҶ

еә”з”ЁзҺҜеўғ

дәҢж¬Ўзҷ»еҪ•пјҢеҚіз¬¬дёҖж¬Ўзҷ»еҪ•еҗҺпјҢе…ій—ӯйЎөйқўпјҢдёҖж®өж—¶й—ҙеҶ…йҮҚж–°и®ҝй—®дёҚйңҖиҰҒйҮҚж–°зҷ»еҪ•

  • еҺҹе®һзҺ°ж–№ејҸпјҡз”ұжңҚеҠЎеҷЁеӯҳеӮЁзҷ»еҪ•дҝЎжҒҜпјҢеҪ“з¬¬дәҢж¬ЎиҜ·жұӮиҝҮжқҘж—¶еҲӨж–ӯд»ҘеүҚжҳҜеҗҰзҷ»еҪ•иҝҮпјҢиӢҘжҳҜеҲҷзӣҙжҺҘи®ҝй—®

    • е®ўжҲ·з«ҜиҜ·жұӮ
    • жңҚеҠЎеҷЁдә§з”ҹзҷ»еҪ•дҝЎжҒҜ
    • зҷ»еҪ•дҝЎжҒҜдј з»ҷе®ўжҲ·з«ҜпјҢе®ўжҲ·з«ҜеӯҳеӮЁеңЁcookieдёӯ
    • е®ўжҲ·з«Ҝз¬¬дәҢж¬Ўи®ҝй—®ж—¶еёҰcookieиҜ·жұӮ
    • жңҚеҠЎеҷЁз«ҜиҝӣиЎҢж ЎйӘҢ

    жіЁпјҡејҠз«ҜпјҢз”ұдәҺзҷ»еҪ•дҝЎжҒҜеӯҳеңЁsessionдёӯпјҢеҚіеӯҳеңЁеҶ…еӯҳдёӯдјҡеҚ з”ЁжңҚеҠЎеҷЁиө„жәҗ

  • зҺ°и§ЈеҶіеҠһжі•пјҡз¬¬дёҖж¬Ўзҷ»еҪ•иҜ·жұӮпјҢжңҚеҠЎеҷЁдә§з”ҹtokenпјҢеӯҳеӮЁе®ўжҲ·з«ҜпјҢз¬¬дәҢж¬Ўи®ҝй—®ж ЎйӘҢtoken

    • е®ўжҲ·з«ҜиҜ·жұӮ
    • жңҚеҠЎз«Ҝдә§з”ҹtoken
    • е®ўжҲ·з«ҜеӯҳеӮЁtoken
    • е®ўжҲ·з«Ҝз¬¬дәҢж¬Ўи®ҝй—®ж—¶еёҰtoken
    • жңҚеҠЎеҷЁиҝӣиЎҢж ЎйӘҢ

и§ЈеҶіз–‘й—®

  • tokenзҡ„еӨҙйғЁе’Ңжңүж•ҲиҪҪиҚ·йҖҡиҝҮbase64еҠ еҜҶе®үе…Ёеҗ—пјҹ

    зӯ”пјҡжӯЈеӣ дёәеҰӮжӯӨпјҢдёҚиғҪеңЁжңүж•ҲиҪҪиҚ·дёӯж”ҫе…Ҙж¶үеҸҠе®үе…Ёзҡ„дҝЎжҒҜеҰӮеҜҶз Ғ

  • жңүж•ҲиҪҪиҚ·иў«зҜЎж”№дәҶжҖҺд№ҲеҠһпјҹ

    зӯ”пјҡз”ұжңҚеҠЎеҷЁдә§з”ҹпјҢз”ұжңҚеҠЎеҷЁж ЎйӘҢпјҢеҚідҪҝtokenеңЁз¬¬дәҢж¬ЎиҜ·жұӮзҡ„ж—¶еҖҷиў«жӢҰжҲӘпјҢз„¶еҗҺеҜ№йҮҢйқўзҡ„жңүж•ҲиҪҪиҚ·иҝӣиЎҢзҜЎж”№пјҢеңЁиҝӣиЎҢж ЎйӘҢзҡ„ж—¶еҖҷпјҢдјҡжҜ”еҜ№зӯҫеҗҚдёІдҝЎжҒҜпјҢиӢҘе°ҶеӨҙйғЁе’Ңжңүж•ҲиҪҪиҚ·д»ҘеҸҠеҜҶй’ҘиҝӣиЎҢеҠ еҜҶеҗҺдёҺзӯҫеҗҚдёІдёҚиғҪеҢ№й…ҚеҲҷиЎЁзӨәиў«зҜЎж”№
