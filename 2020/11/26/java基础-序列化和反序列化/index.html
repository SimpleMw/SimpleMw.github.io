序列化和反序列化

发表于2020-11-26|更新于2021-11-15|java基础相关
|阅读量:

解释：

java核心思想为面向对象，在内存中为一个一个的对象，序列化与反序列化为操作对象的一种方式

序列化：将 java对象 转换为 二进制序列

反序列化：将 二进制序列 还原为 java对象

序列化用途：

  • 对象存储硬盘

  • 对象网络传输
